Analysing the Success of Counter-Strike: A Comparative Study of FPS Games using Player Reviews

This project focuses on the comprehensive analysis and sentiment prediction of user reviews for various video games. The process involved data extraction, cleaning, and visualization of reviews, followed by topic modelling using Latent Dirichlet Allocation (LDA) and sentiment analysis using Natural Language Processing (NLP) techniques. The project effectively models user sentiment, providing valuable insights into user preferences and game performance.
(Softwares and Techniques Used - Python, Jupyter Notebook, LDA, NLP)

Analysing Investment Alternatives for Oil Companies using AHP-GP Programming

This report aims to forecast a profitable investment model for a fictional oil company, Capybarrel Oil Co. Capybarrel Oil Co. has been loosely modelled using figures from Shell’s progress report to make it as realistic as possible. Two multicriteria techniques – Analytical Hierarchy Programming (AHP) and Goal Programming (GP) were applied to this report, by using the weights in the AHP model and incorporating them into a weighted objective function.
(Softwares Used - Python, Microsoft Excel)
